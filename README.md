# 📊 Portfólio de Análise de Dados  

Bem-vindo ao meu portfólio! 🚀  
Aqui compartilho projetos de **Análise de Dados** que desenvolvi com o objetivo de gerar **insights estratégicos** e apoiar a **tomada de decisão**.
